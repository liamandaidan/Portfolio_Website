About

Hello, my name is Liam McLaughlin. I am an aspiring software developer looking to make a name for myself in the industry. I have an eye for detail and have worked in Java since 2013.I have constantly been programming and learning new ways to code. I graduated high school with advanced placement in computer science. I also just graduated with a 2-year diploma in SAIT completed in just over a year. I often find myself working with object-oriented practices and agile development. Software development is a constantly changing industry

Book Store Manager

This program allows customers as well as employees of the ABC bookstore to checkout, find, and list books. The program reads a list of the current books available and sorts them into the following categories: Children's books, cookbooks, paperbacks, and periodicals. The user is then prompted to choose the following options from a menu: checkout book, find books by title, display books by type, and produce a random book list.
